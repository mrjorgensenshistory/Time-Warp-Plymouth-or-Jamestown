## Native-character dignity QC notes

- Powhatan + Massasoit Kevin MacLeod beds (Heavy Heart, Long Note Four) are
  PLACEHOLDERS for the cedar-drum + low-flute brief. Kevin MacLeod's catalog
  does not contain authentic cedar-drum tracks. **Recommended follow-up:**
  source dignified cedar-drum + flute beds from Pixabay
  (search: "native american flute slow", "cedar drum meditation")
  and replace these two slots once auditioned.
- Squanto's "Meditation Impromptu 03" is a piano-flute meditation —
  closer to brief but still a placeholder. Replace with Pixabay
  "wood flute solo" if a better match is found.
